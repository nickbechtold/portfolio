Beach Filters Redesign Live Site The Beach Filters redesign was the first project that I was given full development responisibilities during my time at Pipeline Interactive. I was tasked with building a WordPress website using WooCommerce to categorize and display all of Beach's products. The site is layed out in categories, subcategories and individual product pages with corresponding product tables. I created custom templates and functions using PHP, jQuery, HTML5, and CSS3. Beach Filters - Categories Live Site The architecture of this site involved using the default category and product functions of WooCommerce. By writing custom WP Queries and shortcodes I created four main product categories featured in the main navigation. Each product category then has multiple subcategories. Beach Filters - Subategories Live Site Each subcategory contained indiviudal products. Using Wordpress's built in taxonmies I queried the individual products per subcategory, sorted them based on the client's requirements, and rendered a product attributes table with the default WooCommerce product attributes. Beach Filters - Product Search Live Site Every page of Beach Filters has a form to search the product inventory based on a dynamic dropdown of attributes relating to individual product categories. Specific search options are displayed per each category using jQuery. The client desired specific products to show in multiple search scenarios. Using PHP I wrote my own search algorithm to display the client's desired results.

Radiate (WIP) Read Project Plan Radiate is a personal project / startup of mine that has been turned into a class project consisting of 7 members where I am project manager. Radiate is an online platform for undiscovered designers and musicians to host their work and gain recognition from a community interested in new talent. It began with my roommate and I in 2014 and is still a work in progress. As project manager I collaboratively wrote the project plan, write weekly OKRs, installed GIT and a dev environemt on a virtual server, and oversee the development and marketing activities. The website is being built using WordPress and WooCommerce. We are set to launch a kickstarter campaign in the middle of April 2016.
